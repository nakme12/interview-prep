# Frontend Engineer - Supplier Management Platform

**Company:** TechFlow Solutions
**Location:** Remote / Bangalore, India
**Experience Level:** 1-2 Years
**Salary:** ₹8-12 LPA

---

## About the Role

We are looking for a talented Frontend Engineer to join our growing team and help build scalable, user-friendly interfaces for our enterprise SaaS platform. You will work with modern JavaScript frameworks, collaborate with cross-functional teams, and contribute to features that impact thousands of users daily.

---

## Key Responsibilities

- Develop and maintain frontend features using modern JavaScript frameworks (React, Svelte, Vue)
- Build responsive, performant user interfaces following best practices and design systems
- Collaborate with backend engineers, product managers, and UX designers to deliver high-quality features
- Conduct code reviews and provide constructive feedback to team members
- Manage Git repositories, create pull requests, and resolve merge conflicts
- Participate in sprint planning, daily standups, and retrospectives
- Optimize application performance through techniques like lazy loading, code splitting, and caching
- Write clean, maintainable code following established conventions and patterns
- Debug and troubleshoot issues in production environments
- Implement unit and integration tests for developed features

---

## Required Skills

- **Frontend Technologies:** JavaScript (ES6+), HTML5, CSS3, Responsive Design
- **Frameworks:** React, Svelte, Vue, or similar modern frameworks
- **Version Control:** Git, GitHub, GitLab, pull request workflows
- **Project Management:** Jira, Agile/Scrum methodology, sprint-based development
- **Core Concepts:** Component architecture, state management, API integration, asynchronous programming
- **Development Tools:** VS Code, Chrome DevTools, npm/yarn
- **Soft Skills:** Communication, teamwork, problem-solving, documentation
