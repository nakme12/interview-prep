# Frontend Engineer - Oliv AI

**Job Title:** Frontend Engineer
**Company:** Oliv AI
**Location:** India (Remote)
**Experience Required:** 1-3 years
**Employment Type:** Full-time, Permanent
**Salary:** Competitive with equity

---

## About Oliv AI

Oliv AI is an innovative fintech/AI company building intelligent solutions for the financial services industry.

---

## Key Responsibilities

- Design and maintain engaging frontend experiences
- Build responsive, scalable, and user-friendly web applications, focusing on both performance and usability
- Own components from design to production
- Maintain high code quality and best practices
- Stay current with frontend technology trends
- Translate designs into responsive, performant code
- Collaborate with product managers, designers, and backend engineers
- Participate in code reviews and knowledge sharing

---

## Required Skills & Qualifications

**Technical Stack:**
- TypeScript with strong type system proficiency
- React.js
- Tailwind CSS
- Modern JavaScript tools (Webpack, Babel)
- State management libraries (Redux, Context API, Zustand)
- HTML5, CSS3
- JavaScript ES6+

**Core Competencies:**
- Strong customer-centric mindset (essential)
- Problem-solving abilities
- Eagerness to learn new frameworks and technologies
- Experience building responsive, accessible, high-performance user interfaces
- Understanding of web performance optimization
- Knowledge of version control (Git)

**Nice to Have:**
- Experience with CSS-in-JS libraries
- Testing frameworks (Jest, React Testing Library)
- CI/CD pipelines
- Agile/Scrum methodology experience
- Open source contributions
- Design system experience
