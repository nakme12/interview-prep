# 🎯 Complete FE Interview Roadmap (1+ YOE) - React & Next.js

## 🔥 TIER 1: CRITICAL (Selection depends on these - Spend 70% Time)

### 1. Advanced JavaScript (The Engine)
*   **Topics:** Array Methods (reduce, map, filter), Promises (Promise.all, allSettled), Async/Await, Closures, this keyword (Call/Apply/Bind).
*   **Time:** 5 Hours
*   **Key Question: How do you handle multiple API calls where one might fail but you need the results of the others?

### 2. React Hooks & Lifecycle
*   **Topics:** useEffect (Cleanup, Dependency Array), useMemo & useCallback (Optimization), useRef (DOM access & persistence), useContext (State sharing).
*   **Time:** 6 Hours
*   **Key Question: When would you choose useMemo over a simple variable? Does it always improve performance?

### 3. Next.js Core (The Modern Way)
*   **Topics:** App Router vs Pages Router, SSR (Server Side Rendering), SSG (Static Site Generation), ISR (Incremental Static Regeneration), Client vs Server Components.
*   **Time:** 4 Hours
*   **Key Question: What is the benefit of Server Components (RSC) over standard React components?

### 4. State Management
*   **Topics:** Prop Drilling, Context API vs Redux/Zustand, Data Fetching (React Query / SWR).
*   **Time:** 3 Hours
*   **Key Question: When is Context API NOT enough? Why would you use Zustand/Redux?

---

## ⚖️ TIER 2: IMPORTANT (Spend 20% Time)

### 5. CSS & UI Layouts
*   **Topics:** Flexbox vs Grid, Tailwind CSS basics, Responsive Design (Media queries), Specificity.
*   **Time:** 2 Hours
*   **Key Question: How do you center a div in 3 different ways? How does Flex-grow work?

### 6. Machine Coding & Logic
*   **Topics:** Debouncing (Search bar), Throttling, Flattening Arrays, Removing Duplicates, Infinite Scroll logic.
*   **Time:** 4 Hours
*   **Key Question: Write a custom hook useDebounce from scratch.

### 7. Performance & Optimization
*   **Topics:** Image Optimization (Next/Image), Lazy Loading (React.lazy, Dynamic imports), Code Splitting.
*   **Time:** 2 Hours
*   **Key Question: How do you reduce the bundle size of a React application?

---

## 📝 TIER 3: THEORY & WORKFLOW (Spend 10% Time)

### 8. Web Fundamentals
*   **Topics:** Browser storage (LocalStorage, Cookies), HTTP methods, CORS, JSON Web Tokens (JWT).
*   **Time:** 1 Hour
*   **Key Question: What is the difference between LocalStorage and SessionStorage?

### 9. Version Control & Testing
*   **Topics:** Git Flow (Merge vs Rebase), Basic Unit Testing (Jest/RTL).
*   **Time:** 1.5 Hours
*   **Key Question: How do you handle a merge conflict?

### 10. Next.js Advanced Features
*   **Topics:** Middleware, Image Optimization, SEO (Metadata API), Font Optimization.
*   **Time:** 1.5 Hours
