# 📤 HOW TO PUSH TO GITHUB

## Step 1: Create Repository on GitHub

1. Go to **https://github.com/new**
2. Fill in details:
   - **Repository name:** `interview-prep` OR `pratik-interview-kit`
   - **Description:** "Complete interview preparation kit for Mercanis Frontend Developer role - Resume, technical & behavioral questions"
   - **Public/Private:** Public (better for portfolio)
   - **Add .gitignore:** Already have one
   - **License:** MIT (recommended)
3. Click **Create repository**

---

## Step 2: Add Remote & Push

Copy the commands from GitHub (they'll look like this):

```bash
# Navigate to the folder
cd /Users/mac/Downloads/pratik/interview-prep

# Add remote (replace USERNAME with your GitHub username)
git remote add origin https://github.com/USERNAME/interview-prep.git

# Rename branch to main (if needed)
git branch -M main

# Push to GitHub
git push -u origin main
```

---

## Step 3: Verify on GitHub

1. Go to your repository URL: `https://github.com/USERNAME/interview-prep`
2. You should see:
   - ✅ All markdown files
   - ✅ Resume PDF
   - ✅ README with folder structure
   - ✅ Code with nice syntax highlighting
